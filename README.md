# Prologue

-   rqalpha-mod-xtrade 是 Linux 环境下中国股票市场的交易借口.
-   封装成 [rqalpha](https://github.com/ricequant/rqalpha.git)-mod.
-   支持命令行模式.
-   支持代码中策略实盘模式.
-   无金额大小限制, 1个亿也能玩. 10000000000000000000000000000000
